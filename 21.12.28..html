웹사이트의 완성

a로 링크를 걸어주고 href(html의 reference를 말함)를 걸어준다.


원시 웹

웹 안에 ftb, web 등 여러 서비스들이 있다. 60년에 인터넷 발명, 90년에 웹 발명.
1980년 유럽입자물리연구소에 팀 버너스 리가 취직하고, enquire이라는 프로그램을 만드려 한다.
이 프로그램이 웹의 전신 중 하나이다. 1990년 비로소 연구소에 인터넷이 들어온다.
1990. 10.에 처음으로 웹을 만드는 편집기를 만들고, 11월에 세계 최초로 www를 만들고
12월에 웹 서버를 만들고, info.cern.ch라는 주소를 부여한다.
이렇게 웹이 만들어진 이후로 인터넷이 대중화되는 길이 열린다.


서버와 클라이언트

팀 버너스 리는 두개의 프로그램을 개발하는데, 웹 브라우저와 웹 서버가 그것이다.
웹 서버가 설치 된 컴퓨터에 info.cern.ch주소를 부여받았고, 이 컴퓨터에 index.html이라는 파일이 있다고 하자.
그러면 웹 브라우저가 설치된 컴퓨터가 인터넷에 http://info.cern.ch/index.html을 입력하면
서버컴퓨터가 index.html의 정보를 클라이언트 컴퓨터로 전송해준다. 그럼 웹브라우저 이를 읽어서 화면에 표시한다.
이 때, 웹 브라우저가 설치된 컴퓨터를 Client, 웹 서버가 설치된 컴퓨터를 Sercer로 역할에 따라 달리 부르기로 한다.
자신의 컴퓨터에 서버 파일을 설치할 수도 있고, 다른 회사에 맡기는 호스팀을 할 수도 있다.
호스팅을 이용해 쉽게, 서버를 통해 원리를 파악할 수 있는 것이다.


웹 호스팅

인터넷이 연결된 컴퓨터를 호스트라고 한다. 인터넷을 운영하기 위해 컴퓨터를 빌려주는 회사를 web hosting 업체라 한다.
가장 유명하고 무료인 회사가 Git Hub이다.
로그인 후 new repository를 클릭. 새 저장소를 만드는 과정. 이 때 initialize with READ.ME를 꼭 체크(이유 안알려주심).
깃 허브는 우리가 올려논 파일을 이용해 웹사이트를 이용할 수 있도록 해준다. 그걸 하지 위해 설정-github pages-source에서
Main branch를 선택하고 저장한다. 그러면 웹 사이트 주소가 나온다.
