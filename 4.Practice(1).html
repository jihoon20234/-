2) 문자열 함수 실습

lower() - 소문자로 변환
upper() - 대문자로 변환
strip() - 앞뒤 공백 제거
isalpha() - 문자열이 알파벳으로만 이루져 있는가?
isdigit() - 문자열이 숫자로만 이루져 있는가?
startswith() - 특정한 문자열로 시작하는가?
endswith() - 특정한 문자열로 끝나는가?
find() - 특정한 문자열이 몇번째 인덱스에 있는가?
replace() - 특정한 문자열을 다른 문자열로 바꿀 때 사용


2) 자료구조 실습

- List

Indexing: 대괄호로 위치 지정, 교체 등. Num[0]과 같이 사용
Slicing: :을 사용해 범위 지정. Num[2:-1]과 같이 사용.
In: list 안에 해당 원소가 있는지 묻는 형
append(): list 길이가 1만 늘어나고, 추가되는 자료가 list 형태로 그대로 삽입.
Extend(): 추가하는 자료의 개수만큼 list의 길이가 늘어난다.
Sort(): sort는 정렬하는 함수. Sort(reverse=true)라고 지정하면 정반대의 순서로 정렬. 함수를 적고 shift+Tab을 누르면 대략적인 함수에 대한 설명이 나온다.
Insert(): insert(3, 3)과 같이 입력하면 되고, 3번째 앞에 3이 들어간다.
Remove(): list안의 값 중 가장 앞에 있는 괄호 안의 value 값을 제거.
Del: 지정 위치의 value값을 제거.
Sum/Len(): 앞의 함수들과 다르게 list말고 다른 함수들에도 사용할 수 있다. Sum은 원소들을 다 더한 값, len은 넣은 자료구조의 길이.
Split/Join(): split은 문자열을 나누어 리스트로 만듦, join은 list의 원소를 합쳐 문자열로 만듦. Split은 괄호에 구분자를 넣어주고, join은 join_num = ‘ ‘와 같이 공백을 기준으로 합쳐주는 것.

-Dictionary

속성에 해당하는 값을 알고 싶으면 dict_joe[‘name’]과 같이 속성을 호출하면 대응되는 값이 나온다. 만약 기존에 없는 속성과 값을 입력한 경우 새롭게 dict에 추가된다.
.key(): dict의 키 리스트를 가져온다. 리스트처럼 대괄호로 묶인다.
.values(): dict의 값 리스트. 대괄호.
.items(): dict의 키, 값들의 튜플 리스트를 가져온다.
In 연산자: ‘Joe’ in dict_joe.values()와 같이 딕셔너리 안에 value값, 혹은 key 값의 유무를 물어볼 수 있다.
