Conditional&Loop


-코드 제어의 3가지 구조
: 순차 구조, 선택 구조, 반복 구조가 있다.
순차 구조는 일반적인 구조로서, 입력한 코드를 그대로 실행하는 것.
선택 구조는 조건을 만족했을 때만 선택적으로 실행이 되는 구조.
반복구조는 동일한 명령이 반복되면서 실행되는 구조. 선택구조가 조건문, 반복구조가 반복문.

1)조건문

-If 문의 기본구조
: ‘if condition’의 형태로 쓰이며 condition은 반드시 Boolean형(true/false)이어야 한다.
그래서 condition에는 비교연산자(<,>), 논리연산자(and, or,,)가 주로 사용된다.

-들여쓰기의 의미
: 들여쓰기를 한 문장은 if문의 영향을 받는 것으로 인식한다.
그래서 if문에 해당하는 모든 문장들에는 같은 크기의 들여쓰기를 해야한다.
이는 if 뿐만 아니라 for(반복)/def(함수) 등에도 모두 적용된다.

-조건문 선택
: if문, if-else문, if-elif-else문 등이 있는데 각각 1개, 2개, 3개의 조건을 달 수 있는 것이다.
단순 if문은 조건에 맞을 때만 실행되고 아니면 아무 실행도 하지 않는다.

-‘if-else문’ 기본 구조
: else문은 if문과 별개로 사용할 수 없다.

-‘if-elif-else문’ 기본 구조
: elif는 개수 제한이 없음. 여러 개 사용할 수 있다.
위에서부터 아래로 내려오면서 조건을 충족하는 elif가 있을 경우에는 밑으로 더 내려가지 않는다.


2)반복문

-반복문X자료구조
: 반복문과 자료구조(특히 list)는 잘 어울린다.
목적에 맞게 하나의 속성(엑셀에서의 column)으로 저장된 데이터(나이, 부서이름, 직원이름 등등)가 중요하다.

-For문
: 기본구조는 ‘for 변수 in 리스트’이다.
꼭 리스트가 아닌 이와 유사한 자료구조도 가능(튜플, 문자열 등). 변수 자리에는 자료구조 안 데이터를 어떤 변수로 지정할 것인지 나타낸다.

-For문 사용법
: any_list = [0, 1, 2, 3, 4]
For val in any_list:
Print(val)
을 입력하면 5번 반복하면서 0, 1, 2, 3, 4가 순서대로 출력된다. 리스트의 길이가 5이기 때문에 총 5회 반복하는 것.

-for문 X 리스트
: 리스트와 for문을 이용해서 새로운 값으로 채워진 list 만들기
List_upper = [‘A’, ‘B’, ‘C’] -> 3회 반복임을 알 수 있다.
List_lower = [ ]
For alp in list_upper:
List_lower.append(alp.lower())
Print(list_lower)
를 입력하면 첫 반복때는 list_lower에 ‘a’가 추가되고, 세번째 반복 때는 alp가 ‘C’를 가리키고, ‘c’가 list_lower에 추가된다.

-for문 – range()함수
: range함수는 등차수열로 값이 채워진 리스트와 유사한 형태의 자료구조를 만드는 함수. Slicing과 굉장히 유사.
input으로는 stop, start, step이 있고, 각각 그 전에 마침, 그 숫자에서 출발, 그 숫자가 등차인 것을 의미한다.
Range(5)의 경우 input이 하나이므로 5가 stop을 의미하고 원소로는 0~4이므로 5회 반복실행하는 것이다.
-Skip, Stop
For문 도중 continue를 만나면 나머지 코드 skip하고 다음 반복문으로 진행하고, break는 아예 빠져나오면 된다. Continue는 해당 iteration만 빠져나온다.


3) 함수

-함수의 구조(1)
Def func_name(param1, param2):
Val = algorithm
Return value
의 구조. Def는 함수를 정의하겠다는 뜻이고, func_name에는 함수의 이름을 적어준다.
Param1, param2 등은 input이다. Val = algorithm에는 함수가 수행할 일을 적어준다.

하나 더 보자.

Def square(x)
Y = x*x -->선언
Return y
C = square(3) --> 호출
의 구조. 선언 시 square의 함수는 변수 x를 input으로 갖게 되고, 호출 시 입력한 3은 자동적으로 x = 3의 구조를 거친다.
return되는 output 값은 선언 시 지정한 알고리즘에 따라 y = x*x를 거쳐 결과값이 나온다.
결국 c = 9로 연산이 일어난다. C에 9가 할당되는 것.
만약 c라는 변수에 할당시키지 않았다면 그냥 내부적인 계산만 일어나고 아무런 소용이 없다. 추후 사용할 경우 변수(c 등)에 꼭 할당시키기.

-함수의 구조(2)
: 두 값을 더하는 함수.
Def sum_two(a, b):
Val = a+b --> 선언
Return val
C = sum_two(3, 12) --> 호출
의 구조. Input이 전과 다르게 두개인 것을 확인. 두 변수 중 먼저 쓰인 것을 a, 나중을 b라 한다.
그리고 output값은 val로 할 것임을 선언하고 있다. 호출 과정에서 C에는 15가 할당된다.

-함수의 구조(3)
: 이번 함수는 input값이 필요없는 함수. 따라서 output도 없다.
Def hello_world(): -->선언
Print(“hello, world!”) Hello_world() -->호출
의 구조. 알고리즘도 없고, 그냥 출력만 한다.

Print(“x = “ + str(x))
Func(5)
